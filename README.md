# pokemon_home_SV
全国図鑑を生成する
```
python create_zukan.py
```

ランクマッチ使用率の一覧を生成する
```
python create_battle_data.py
```

日本語 <-> 外国語名の対応表を生成する
```
python create_translation_table.py
```

alias.py
ポケモンのエイリアス名を返す関数を格納. Import用

##　出力形式
json/csv
